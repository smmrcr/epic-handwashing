EPIC HAND WASHING IN A TIME OF LOST NARRATIVES is an iteration on the project A Kitchen of One's Own, developed for the media wall in the Photographer's Gallery (London). This iteration of the project displays a set of 74 videos from the EPIC KITCHENS dataset that were trained for machine learning with the verb "wash" and the paired noun "hand".
